# SmileBrightbase

A dental clinic booking system web application built with HTML, CSS, JavaScript, and PHP.

## Features

- Online appointment booking system
- Dentist selection and scheduling
- Booking management and updates
- Server-side form processing
- Database integration (MySQL)
- Responsive design

## Requirements

- PHP 7.4 or higher
- MySQL/MariaDB
- Apache web server (XAMPP recommended)
- Modern web browser

## Setup

1. Clone the repository
2. Configure your database settings in `src/config/database.php`
3. Run the database migrations from `database/migrations/`
4. Start your Apache and MySQL services
5. Access the application at `http://localhost/SmileBrightbase/public/`

## Project Structure

- `public/` - Public-facing HTML files
- `api/` - PHP API endpoints
- `src/` - Source code (config, models, services)
- `database/` - Database migrations and seeds
- `templates/` - Email templates
